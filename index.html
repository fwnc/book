JWD的python自学教程

JWD的python自学教程

简要说明

本教程来自网络资源，主要涵盖了python的基础语法体系，本人将其文档以GitBook形式整理，并上传至GitHub个人网页进行托管，以方便本人随时查看。

主要内容

序号 内容 目标
01 Linux 基础 对 Ubuntu 的使用从很陌生 达到灵活操作
02 Python 基础 涵盖 Python 基础知识，掌握基础的编程能力
03 Python 面向对象 介绍 Python 的面相对象开发，为开发大型项目做好铺垫和准备
04 项目实战 应用基础班学习过的知识，编程实战，完成第一个 Python 项目

GitBook目录链接不能跳转的解决方法

对Gitbook生成文件夹_book/gitbook/theme.js的文件进行修改，if(m)for修改为if(false)for
